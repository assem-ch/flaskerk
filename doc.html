class flaskerk.exception.HTTPException(code: int = Schema(default: Ellipsis, description: 'HTTP response status code', gt: 100, lt: 1000, extra: {}), msg: str = None)[source]¶

HTTP Exceptions.

Parameters

  • code (int) – HTTP status code

  • msg (str) – description

If you are using default code like 403, 404, 500 …, you can ignore the msg parameter and this module will use default description.

If you want to define your own code and description, you must offer msg.
